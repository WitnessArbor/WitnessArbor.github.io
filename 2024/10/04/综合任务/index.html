程序安装

More info: 下载并安装合适的 nodejs 和 git。

More info: 1/对前期（初学者）而言，配套程序影响不大。
2/13.14版本，其配套程序，于 Window 7 而言，大部分不是最后一个支持的版本。
3/ nodejs 的最新版本和最新的 LTS 版本 v20.18.0 （目前本机使用），安装配套程序时（本机），部分指令会报错，使过程复杂化，不过可以先安装会报错的（安装指令中有检测的步骤），来解决这个问题。
4/安装 phthon 时，不会生成 path 参数，不利于后面的使用。

Github账号注册及相关步骤

More info: 1/用户注册。
2/创建仓库（格式：用户名.github.io），注意选择 Public 。
3/进入仓库，选择 Code ，选 ssh ，复制地址。

More info: （以下攻略适用到2024.11.11，还有新手不要随便安装插件，官方的也不行！！！）

安装过程

本地安装

首先在磁盘中创建一个用来存放Github本地仓库文件的目录blog，右键 git bash 打开。

More info: 1/输入指令 npm install -g hexo-cli
2/输入指令 npm hexo init
3/输入指令 hexo s ,访问网站 localhost:4000
4/输入指令 cd ~/.ssh
5/输入指令 ssh-keygen -t rsa -C ‘注册时的邮箱地址’ （要回车几下）

配置仓库

More info: 1/查看C盘目录(此文件夹所在C:\用户\用户名目录下)，找到刚刚生成的 .ssh 文件夹。
2/在记事本中打开 id_rsa.pub(网上有人说是id_rsa.put,看情况吧) 文件并复制生成的 key（全部）
3/在GitHub主页的头像下找到 seething，然后点击 SSH and GPG keys，点击 New SSH key
4/将刚刚在 ssh 目录下所复制的 key 复制进 Key 这个空(注意空格)（名字随意）。
5/在 git bash 下输入指令 ssh -T git@github.com（最后一次显示拒绝，显示22端口拒绝访问，要用这个指令 ssh -p 443 -T git@ssh.github.com 强制访问443端口）。
6/在 ？ 后输入 yes ,出现用户名成功。
7/输入指令 git config –global user.name “注册时用户名” 和 git config –global user.email “注册时邮箱” 进行绑定。

收尾工作

More info: 打开 blog 底下的 _config 文件，将末尾的代码改成下文：
deploy:
type: git
repo: git@github.com:用户名/用户名.github.io.git
branch: main
这样，博客就搭好了。（上传方法，请参照）
